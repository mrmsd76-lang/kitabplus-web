# KitabPlus Web Application

This is the web version of the KitabPlus mobile application, built with Expo and React Native Web.

## Features

- 📱 Responsive design for desktop and tablet
- 📚 Complete book library with summaries, quizzes, and challenges
- 💳 Integrated payment system (Wise)
- 👤 User authentication and profiles
- 📊 Admin dashboard for book management

## Deployment

This site is deployed on Cloudflare Pages and automatically updated when changes are pushed to the main branch.

## Local Development

```bash
# Install dependencies
pnpm install

# Start development server
pnpm run dev

# Build for production
pnpm run build

# Export web version
pnpm expo export --platform web
```

## Technology Stack

- React Native Web
- Expo Router
- NativeWind (Tailwind CSS)
- TypeScript
- Wise Payment Integration
